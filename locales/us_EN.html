User Manual Simple PDF Editor v.0.1a

This application is designed for PDF files simple manipulation.

Application capabilities:

- rotate selected pages;

- delete selected pages;

- export selected pages to separate JPG files;

- import selected pages from another PDF file;

- import selected images as pages of PDF document;

- rearrange selected pages in PDF file;

- insert numeration on selected pages of PDF file;

- multiple PDF files compression;

- multiple PDF files merging;

- multiple BMP, PNG and JPG files merging in one PDF file.

Application menu consists of:

- File

- View

- Help

Submenu File actions:

- Open – opens one or multiple PDF files for editing. Pages of opened PDF files converts to images, at this point editing of page content in unavailable.

Available operations with opened PDF files:

Note: for all page operations you can choose multiple page while hold “Ctrl” key.

- Selected pages rotation – you can rotate selected pages clockwise (action “Rotate CW”) or counterclockwise (action “Rotate CCW”)

- Selected pages removal– you can delete selected page (action “Delete”)

- Selected pages JPG export – you can export selected pages as separate JPG files (action “Pages extract”). Exported pages stored in the subdirectory named same as opened PDF file, located near the opened file. Subdirectory contains JPG files named as: “name_of_PDF_file_[page number].jpg”

- Insert selected pages from another PDF file – after execute shows file dialog to choose PDF files for page selection (action “Pages insert”). For selected file preview pages dialog executes, after that you can choose which pages will be imported. Also you can choose before or after selected page this pages will be inserted.

- Import selected images as pages of PDF document - after execute shows file dialog to choose images to import. Supported image formats: BMP, PNG, JPG. For selected files preview pages dialog executes, after that you can choose the order of import images. Each image imported as separate PDF document page. Also you can choose before or after selected page this images will be inserted.

- Rearrange pages in PDF file – shows page preview dialog to choose before or after selected page move needed pages (action “Move pages”)

- Insert page numbers – inserts page number in bottom right corner of page after executed (action “Insert page numbers”)

- File compress – reduces size of PDF document by ratio, selected near button “Compress document”

- Document save – you can choose the document to save current changes – after execution of “Save” action changes stored in current PDF file, or you can choose separate file when execute “Save as” action.

- Merge PDF files – can merge selected PDF files into one. After executed shows first page preview dialog, where you can set the order of merging. Move left or right performs by click “<” or ”>” buttons. After “Merge” button clicked file dialog executes to set the name for PDF file, contains merged files. Files merged in left to right order, final file automatically opens.

- Compress PDF files – After executed shows dialog to set Compress level – digit from 0 to 4, where 0 – minimum and 4 – maximum compress ratio. Default Compress level is 2. After change of Compress level automatically calculates size of compressed file and shows in dialog.

Note – compressing works only for PDF contains images (for example, scanned docs).

Typically the bigger Compress level is set the smaller final file result. After “Compress” button clicked selected files are compressing by selected Compress level and opens in application. Note – after these file opened as temporary files, to save them you must use action “Save as” (action “Save” saves it in Temp folder of your profile). If you check the “Auto file names” box – compressed files automatically creates in the directory same as opened files and they named as “File name_compressed_[ Compress level].pdf”.

- Merge images to PDF – opens multiple image files to merge in one PDF document. Each selected image file converts to separate PDF file page. Supported image formats: BMP, PNG, JPG. When select files you can choose different formats simultaneously. After executed shows image preview dialog, where you can set the order of pages in merged file by click “<” or ”>” buttons. After clicked “Merge” button selected images merges in one PDF file in selected order left to right. Final PDF file name is set in file dialog after executing of “Merge” action and this file automatically opens in application.

Submenu “View” contains:

- Language

- Theme

Submenu “Language” for manual localization set, if operating system locale is not supported by application – default “English” locale is used.

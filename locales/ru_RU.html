Руководство пользователя Simple PDF Editor v.0.1a

Приложение предназначено для простых манипуляций с файлами формата PDF.

Список операций:

- поворот выбранных страниц;

- удаление выбранных страниц;

- экспорт выбранных страниц в виде отдельных файлов формата JPG;

- вставка выбранных страниц из другого PDF файла;

- вставка изображений в качестве страниц;

- изменение порядка следования выбранных страниц в документе;

- вставка нумерации на выбранные страницы;

- сжатие одного или нескольких PDF файлов;

- объединение нескольких PDF файлов в один файл;

- объединение нескольких файлов изображений форматов BMP, PNG и JPG в один файл.

Меню приложения состоит из подменю:

- Файл

- Вид

- Помощь

Подменю «Файл» содержит действия:

- Открыть – открывает для отображения и редактирования один или несколько PDF файлов. Открываемые страницы файлов отображаются в виде изображений, редактирование текста на данный момент недоступно.

После открытия документа на данный момент поддерживаются следующие операции:

Обратите внимание: для всех операций над страницами допустим множественный выбор при нажатой клавише «Ctrl»

- Поворот выбранных страниц – после выбора необходимых страниц, их можно повернуть по часовой (действие «Поворот вправо») или против часовой стрелки (действие «Поворот влево»).

- Удаление выбранных страниц – после выбора необходимых страниц, их можно удалить из документа (действие «Удалить»).

- Экспорт выбранных страниц в виде отдельных файлов формата JPG – после выбора необходимых страниц, их можно сохранить в виде отдельных файлов изображений (действие «Извлечь страницы»). После запуска этой операции в каталоге, содержащем открытый PDF файл создается каталог с именем файла PDF, внутрь которого помещаются выбранные страницы, с наименованием формата «имя_файла_[номер страницы].jpg».

- Вставка выбранных страниц из другого PDF файла – вызывается диалог выбора файла, из которого требуется вставить страницы (действие «Вставить страницы»). После выбора файла отображается диалог предпросмотра страниц этого файла, после чего необходимо выбрать страницы, которые будут помещены в исходных документ. Также можно выбрать до или после указанной страницы будет осуществляться вставка страниц.

- Вставка изображений в качестве страниц - вызывается диалог выбора файлов, на данный момент поддерживаются файлы формата BMP, PNG, JPG. После выбора файлов отображается диалог их предпросмотра, в котором можно указать порядок следования импортируемых изображений. Каждое изображение импортируется как отдельная страница PDF документа. Также можно выбрать до или после указанной страницы будет осуществляться вставка изображений.

- Изменение порядка следования выбранных страниц в документе – после запуска отображается диалог, в котором можно выбрать страницу, а также порядок – до или после нее переместить выбранные страницы (действие «Переместить страницы»).

- Вставка нумерации на выбранные страницы – после запуска в правом нижнем углу выбранных страниц будет вставлен их номер (действие «Вставить номера страниц»).

- Сжатие документа – происходит сжатие документа с уровнем сжатия, указываемым рядом с кнопкой «Сжать документ».

- Сохранение документа – происходит либо сохранение изменений в текущий открытый документ – при выборе действия «Сохранить», либо в файл с другим именем, указываемым в диалоге – при выборе действия «Сохранить как».

- Объединить PDF файлы – открывает несколько PDF файлов для объединения в один. После запуска отображает диалог, в котором можно указать порядок следования PDF документов в конечном документе нажатием кнопок «<» и «>». В диалоге отображается имя и предпросмотр первой страницы файлов PDF, которые будут объединены в один документ. После нажатия кнопки «Объединить» файлы в указанном порядке следования слева-направо будут объединены в один файл формата PDF, имя конечного файла задается в диалоге, вызываемом после нажатия кнопки «Объединить». После указания имени конечного файла происходит объединение документов и открытие этого файла.

- Сжать PDF файлы - открывает несколько PDF файлов для уменьшения их размера. После запуска отображает диалог, в котором можно указать Уровень сжатия. Уровень сжатия – число от 0 до 4, где 0 – минимальное сжатие, 4 – максимальное сжатие. По умолчанию установлена степень сжатия 2. При изменении уровня сжатия происходит автоматический пересчет размера сжимаемых файлов и отображение этого размера с указанием степени уменьшения размера в процентах.

Обратите внимание – сжатие работает только для PDF документов, содержащих растровое изображение (например, сканированные документы).

Отображаемая в процентах степень сжатия может иметь отрицательные значения – в таком случае конечный документ увеличивается в размерах, для уменьшения размера необходимо увеличить Уровень сжатия. После нажатия кнопки Сжать происходит сжатие файлов с указанным Уровнем сжатия и открытие данных документов, данные документы открыты в виде временных файлов, для сохранения их в таком виде необходимо использовать операцию «Сохранить как» для каждого документа. При отметке чекбокса «Автоматические имена файлов» сжатые файлы создаются автоматически и помещаются в те же каталоги, из которых они были открыты и получают имя в виде «Исходное имя_сжатый_[Уровень сжатия].pdf».

- Объединить изображения в PDF - открывает несколько файлов изображений для объединения в один файл формата PDF. Каждый выбранный файл преобразовывается в отдельную страницу в конечном PDF файле. На данный момент поддерживаются файлы формата BMP, PNG, JPG. При выборе объединяемых изображений допустим одновременный выбор файлов разных поддерживаемых форматов. После запуска отображается диалог, в котором отображен предпросмотр объединяемых файлов и можно указать порядок следования страниц в конечном документе нажатием кнопок «<» и «>». После нажатия кнопки «Объединить» файлы изображений в указанном порядке следования слева-направо будут объединены в один файл формата PDF, имя конечного файла задается в диалоге, вызываемом после нажатия кнопки «Объединить». После указания имени конечного файла происходит объединение изображений и открытие этого файла.

Подменю «Вид» содержит подменю:

- Язык

- Тема

Подменю «Язык» необходимо для выбора языка локализации приложения, при запуске происходит определение текущей локали операционной системы и установка языком приложения языка локали, при условии, что язык локали поддерживается приложением. Если язык локали не поддерживается приложением – языком приложения устанавливается английский.

На данный момент поддерживаются языки:

- Английский

- Русский

Подменю «Тема» изменяет цветовую схему приложения.
